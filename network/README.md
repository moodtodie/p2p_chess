### Подход "одноранговая сеть"

В подходе "одноранговая сеть" (P2P) игроки соединяются друг с другом напрямую, без использования сервера.

### Структура пакета

| dst_key | src_key | is_service | data | checksum |
|---|---|---|---|---|

* dst_key - уникальный ключ получателя (создается при начале партии)
* src_key - уникальный ключ отправителя (создается при начале партии)
* is_service - флаг служебного пакета
* data - полезные/служебные данные
* checksum - контрольная сумма (SHA-256)

### Структура полезных данных

| player | pos1 | pos2 | num_of_turn |
|---|---|---|---|

* player - Имя пользователя совершившего ход
* pos1 - Позиция с которой фигура ходит
* pos2 - Позиция на которую фигура ходит
* num_of_turn - Номер хода
